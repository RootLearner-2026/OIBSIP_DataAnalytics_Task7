## Oasis Infobyte Internship
Task 5 – Data Analytics

### Objective
The objective of this project is to build a machine learning model to detect fraudulent credit card transactions.

### Dataset
This dataset contains credit card transactions made by European cardholders in September 2013.

Total Transactions: 284,807  
Fraud Cases: 492  

Dataset Source:  
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

### Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

### Steps Performed
1. Data Loading
2. Data Exploration
3. Train-Test Split
4. Model Training using Random Forest
5. Fraud Prediction
6. Model Evaluation
7. Confusion Matrix Visualization

### Model Used
Random Forest Classifier

### Results
Accuracy achieved: **99.95%**

The model successfully detects fraudulent transactions with high precision.

### Conclusion
Machine learning models can effectively identify fraudulent credit card transactions and help financial institutions prevent financial losses.
